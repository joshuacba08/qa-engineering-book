# QA Engineering: Fundamentos Profesionales del Testing Moderno

Este libro está diseñado para testers con experiencia práctica que desean consolidar los fundamentos teóricos del aseguramiento de calidad de software.
Combina la claridad conceptual de los textos de _Effective Software Testing_ y _Software Testing Strategies_ con un enfoque moderno, aplicado a la práctica real del QA Engineer.

---

## Objetivos

- Sanar las lagunas teóricas y metodológicas comunes en testers empíricos.
- Enseñar a **diseñar, ejecutar y automatizar pruebas** con criterio técnico.
- Conectar el testing con **Scrum, CI/CD y buenas prácticas de ingeniería de software**.
- Convertir el conocimiento disperso en una **guía estructurada, práctica y moderna**.

---

## Estructura

El libro está dividido en 5 partes:

1. **Pensar como un QA** — fundamentos, riesgo y diseño de tests.
2. **Técnicas clave** — especificaciones, cobertura, propiedades y contratos.
3. **Automatización inteligente** — frameworks, APIs y buenas prácticas.
4. **Estrategia y cultura de calidad** — QA dentro del ciclo de desarrollo.
5. **Profesionalización y futuro** — ética, AI y visión avanzada.

---

## Requisitos

- Conocimientos básicos de software o experiencia en testing manual.
- Familiaridad con herramientas como Postman o algún lenguaje de scripting.

---

## Tecnologías y ejemplos usados

- **Lenguajes:** JavaScript, Python, TypeScript.
- **Frameworks:** Playwright, Jest, JUnit, Pytest.
- **CI/CD:** GitHub Actions, Jenkins, Docker.
- **Diagramas:** Mermaid.
- **Formatos:** Markdown, JSON, YAML.
